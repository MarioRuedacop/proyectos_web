Conceptos básicos de programación web


Arquitectura cliente servidor

La arquitectura cliente-servidor es un modelo donde un cliente (como el navegador) envía solicitudes a un servidor, que procesa datos y responde con la información solicitada.

El desarrollo web, comenzó con páginas estáticas (solo HTML), luego pasó a sitios dinámicos con servidores que generan contenido (PHP, bases de datos) y hoy incluye aplicaciones modernas con APIs, JavaScript avanzado y arquitecturas distribuidas (como SPA y microservicios).


Uso básico de etiquetas HTML

Las etiquetas HTML son elementos que definen la estructura y el contenido de una página web. Algunas etiquetas comunes incluyen:
  • <h1> a <h6>: Encabezados de diferentes niveles.
  • <p>: Define un párrafo.
  • <br>: Crea un salto de línea.
  • <img>: Inserta una imagen.
  • <ul> y <ol>: Listas desordenadas y ordenadas, respectivamente.
